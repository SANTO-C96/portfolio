Project 4


Temperature Convertor


Description


Designed and developed a Temperature Converter web application using HTML, CSS, and JavaScript. This application allows users to convert temperatures between Celsius, Fahrenheit units seamlessly. The user-friendly interface and intuitive design ensure a smooth and efficient temperature conversion experience.


Key Features:


  • Interactive Interface: Created an engaging user interface with HTML and CSS that ensures a visually appealing and user-friendly experience.
  • Real-time Conversion: Implemented JavaScript functionality to perform temperature conversions instantly as users input values, without requiring page reloads.
  • Unit Selection:Provided users with the option to select the input and output temperature units, allowing for conversions across Celsius, Fahrenheit, and Kelvin scales.
  • Tools Used: -HTML, CSS, JAVASCRIPT
